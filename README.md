## General info
This project is ToDo List(Tasks tracker)
	
## Technologies
Project is created with:
* Angular CLI: 13.2.3
* Node: 14.17.6
* json-server@0.17.0
	
## Setup
To run this project, install it locally using npm:
* npm run serve
* ng serve
* npm install -g json-server
* json-server --watch db.json
